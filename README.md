## 汉化

### Windows
**app.rar|zip**  
解压&&复制app文件夹到resources目录下即可  
~/Postman/app-\*.\*.\*/resources/  
(PS:~为应用程序安装目录，默认在C:/Users/用户名/AppData/Local/Postman)

**app-\*.\*.\*.rar|zip**  
解压直接使用

### Mac
解压&&替换app目录  
目录位置 ~/Postman.app/Contents/Resources/app

### Linux
解压&&替换app目录
