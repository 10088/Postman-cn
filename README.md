## 安装

### Windows
**app.rar|zip**  
解压&&复制app文件夹到resources目录下即可(请将解压的node项目放在app文件夹里面)  
~/Postman/app\*.\*.\*/resources/

**app\*.\*.\*.rar|zip**  
解压直接使用

### Mac
解压&&替换app目录
目录位置 ~/Postman.app/Contents/Resources/app

### Linux
解压&&替换app目录
